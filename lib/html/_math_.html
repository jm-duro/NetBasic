function

Following global functions are defined:

  • is_in_range (atom, atom, atom)

    Params:
    • atom item

      item to check

    • atom from

      minimal value

    • atom upto

      maximal value

    Return:

    1 if within range, 0 if not

    Example:
    i = is_in_range(3.14, 3, 4)
    i is 1
    i = is_in_range(3.14, 2, 3)
    i is 0

  • is_powerof2 (object)

    Params:
    • object p

    Description:

    may be applied to an atom or to a sequence

    Example:
    for i = 1 to 5 do
    ? {i, powof2(i)}
    end for
    output ...
    {1,1}
    {2,1}
    {3,0}
    {4,1}
    {5,0}
