  • replace_text (sequence, sequence, sequence, sequence)

    Params:
    • sequence old_file

    • sequence new_file

    • sequence old

      text to be replaced

    • sequence new

      new text

    Description:

    result is written in a new file
    old file is not affected

    Example:
    replace_text("test_file_.bak", "new_file_.bak", "this", "that")

  • set_param_value (sequence, sequence, sequence, integer)

    Params:
    • sequence list

    • sequence param

      parameter name

    • sequence value

      parameter value

    • integer add

      indicates to add value if not already existing. defaults to 1

    Return:

    sequence: modified list

    Description:

    if add is false (0), parameter value must exist to be modified
    if add is true (1), a non-existing parameter will be added

    Example:
    sequence params
    params = {"first=1","second=2","third=three"}
    analyze_object(set_param_value(params, "third", "3"), "set_param_value")
    set_param_value =
    . [1] "first=1"
    . [2] "second=2"
    . [3] "third=3"
    analyze_object(set_param_value(params, "fourth", "4", 0), "set_param_value")
    set_param_value =
    . [1] "first=1"
    . [2] "second=2"
    . [3] "third=three"
    ("add" is not set so parameter "fourth" is not added)
    analyze_object(set_param_value(params, "fourth", "4"), "set_param_value")
    set_param_value =
    . [1] "first=1"
    . [2] "second=2"
    . [3] "third=three"
    . [4] "fourth=4"

    See also:

    find_param_name(), get_param_value()

  • read_ini_file (sequence)

    Params:
    • sequence filename

    Description:

    result is a sequence { {section name, section content}, ... }
    each section content is a list of parameters and values { "parameter=value", ... }

    Example:
    file test.ini contains following lines:
    [Telnet]
    host=127.0.0.1
    port=23

    [SSH]
    host=127.0.0.1
    port=22

    [FTP]
    host=127.0.0.1
    port=21

    analyze_object(read_ini_file("test.ini"), "read_ini")
    read_ini =
    . [1]
    . . [1] "Telnet"
    . . [2]
    . . . [1] "host=127.0.0.1"
    . . . [2] "port=23"
    . [2]
    . . [1] "SSH"
    . . [2]
    . . . [1] "host=127.0.0.1"
    . . . [2] "port=22"
    . [3]
    . . [1] "FTP"
    . . [2]
    . . . [1] "host=127.0.0.1"
    . . . [2] "port=21"

    See also:

    write_ini_file()

  • get_ini_section (sequence, sequence)

    Params:
    • sequence ini

      INI file content

    • sequence section

      section name

    Return:

    sequence: selected section content

    Description:

    file content has to be read previously with read_ini_file()

    Example:
    ini = read_ini_file("test.ini")
    s = get_ini_section(ini, "SSH")
    s may be (example):
    . [1] "host=127.0.0.1"
    . [2] "port=22"
