function

Following global functions are defined:

  • md5 (sequence)

    Params:
    • sequence message

      string to compute

    Return:

    sequence of 16-byte hash

    Example:
    whole_file = read_file(filename)
    return hex_string(md5(whole_file))

    See also:

    md5sum()

  • md5sum (sequence)

    Params:
    • sequence filename

      the filename to compute

    Return:

    sequence: an hexadecimal MD5 hash string of the file content

    Example:
    s = md5sum(sequence filename)

    See also:

    md5()
