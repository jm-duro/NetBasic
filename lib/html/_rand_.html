_rand_

procedure

Following global procedures are defined:

  • initialize ()

    Example:
    initialize()

    See also:

    randomize()

  • randomize ()

    Description:

    The seed provided to the random number generator is an atom
    calculated from a high-resolution time factor so it is very unlikely
    to be the same at next launch

    Example:
    randomize()

    See also:

    initialize()
