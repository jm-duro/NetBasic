_ssh2_

constant

Following global constants are defined:

  • TCP_HOST   [ 1 ]

  • TCP_PORT   [ 2 ]

  • TCP_SOCKET   [ 3 ]

  • SSH_SESSION   [ 4 ]

  • SSH_STATE   [ 5 ]

  • BUFFER   [ 6 ]

  • DISCONNECTED   [ 0 ]

  • CONNECTED   [ 1 ]

variable

Following global variables are defined:

  • sequence activeConnections

    list of active connections
    active does not mean connected (see activeSockets for that)

  • integer rtn_ident_cnx

    id of the routine that identifies items in activeConnections
    to be used for debug with analyze_object
    example:
    analyzeObject(activeConnections, "activeConnections", f_debug, rtn_ident_cnx)

procedure

Following global procedures are defined:

  • ssh_exit ()

    Exit the libssh2 functions and free's all memory used internally.

    Example:
    ssh_exit()

    See also:

    ssh_init()

  • ssh_close (integer)

    close a SSH connection

    Params:
    • integer cnx

      connection id

    Description:

    updates activeConnections

    Example:
    ssh_close(1)

    See also:

    ssh_connect()

  • ssh_exec (integer, sequence, integer)

    run a command on the SSH session

    Params:
    • integer cnx

      connection id

    • sequence cmd

      command to run

    • integer terminal

      opens a terminal if 1 (true) (defaults to 0)

    Example:
    ssh_exec(1, "ls -l", 1)

function

Following global functions are defined:

  • check_link (integer)

    check if a connection is valid

    Params:
    • integer cnx

    Example:
    if not check_link(1) then puts(2, "Connection 1 failed!") end if

  • get_buffer (integer)

    reads connection buffer

    Params:
    • integer cnx

      connection id

    Return:

    sequence: buffer content

    Example:
    data = get_buffer(cnx)

  • set_buffer (integer, sequence)

    sets connection buffer

    Params:
    • integer cnx

      connection id

    • sequence data

      new buffer content

    Return:

    integer: 1 on success, 0 on failure

    Example:
    data = get_buffer(cnx)
    ok = set_buffer(cnx, remove(data, 1, 7))

  • clean_buffer (integer)

    cleans connection buffer

    Params:
    • integer cnx

      connection id

    Return:

    integer: 1 on success, 0 on failure

    Example:
    ok = clean_buffer(cnx)

  • find_connection (sequence, integer)

    find a connection id from host and port used

    Params:
    • sequence host

    • integer port

    Example:
    i = find_connection("192.168.1.250", 23)

  • ssh_init ()

    Initialize the libssh2 functions

    Return:

    Returns 1 if successful else returns 0

    Example:
    if not ssh_init() then abort(1) end if

    See also:

    ssh_exit()

  • ssh_connect (sequence, integer, sequence, sequence)

    open a SSH connection to a remote host

    Params:
    • sequence host

    • integer port

    • sequence username

    • sequence password

    Return:

    integer: connection id

    Description:

    updates activeConnections

    Example:
    cnx = ssh_connect("192.168.1.250", 22, "admin", "admin")

    See also:

    ssh_close()
